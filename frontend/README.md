# 🌾 Farmer AI Assistant

AI-powered mobile app to help farmers detect crop diseases using images.

## Features
- 📸 Upload crop image
- 🧠 AI disease detection
- 🇮🇳 Hindi responses

## Tech Stack
- Flutter (Frontend)
- FastAPI (Backend)
- TensorFlow (AI)

## Setup

### Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

### Frontend
cd frontend
flutter pub get
flutter run
